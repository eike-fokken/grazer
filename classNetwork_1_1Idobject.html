std::unique_ptr<std::string const> const Network::Idobject::idptr
private

pointer to the actual string holding the id.

This is a unique pointer instead of the string itself to save space on the stack. The price for dereferencing is acceptable because the id is used only during construction of a network and during writing out the data to the output files.
