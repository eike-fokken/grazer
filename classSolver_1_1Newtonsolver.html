Detailed Description

Manages solving non-linear systems and (to be implemented) computing derivatives with respect to controls.

This class holds a SparseMatrix and a corresponding Sparse-matrix solver, so it can compute the solution of a non-linear problem.

This method computes a solution to f(new_state) == 0.

It uses an affine-invariant Newton method described in chapter 4.2 in "Deuflhard and Hohmann: Numerical Analysis in Modern Scientific Computing". Afterwards there should hold f(new_state) == 0 (up to tolerance).
