Detailed Description

A base class for network components that supply model equations to be solved.

An equation component owns a number of indices, namely between #start_state_index (inclusive) and #after_state_index (exclusive), which will be filled with results of the model equations of the component. It is implicitely taken for granted that the set of owned indices of all objects starts at 0 and is consecutive.
