Detailed Description

template<typename ConcreteNode>
struct Model::Componentfactory::NodeType< ConcreteNode >

A NodeType is an instantiation of the class of a (concrete) node. It allows the passing and storage of NodeTypes which are all subclasses of AbstractNodeType. And the AbstractNodeType interface provides dynamic access to the static methods of (concrete) nodes.

Their main use is for dynamic configuration of the Componentfactories by adding NodeTypes to their node_type_map. They also enable the creation of JSON Schemas describing the data needed for the instantiation of the (concrete) nodes.

Preconfigured Componentfactories are found in Full_factory, Gas_factory and Power_factory
