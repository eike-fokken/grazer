Set the derivatives with respect to the boundary states.

If (p,qvol) = phi(rho,q), then this function must compute

dF/dState = function_derivative*phi'

and call

jacobianhandler->set_coefficient(rootvalues_index, rho_index, dF/dState[0]);
jacobianhandler->set_coefficient(rootvalues_index, q_index, dF/dState[1]);
