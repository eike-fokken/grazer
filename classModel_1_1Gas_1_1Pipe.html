derivative of Equationcomponent::evaluate w.r.t. last_state.

evaluates the derivative of Equationcomponent::evaluate and hands the result to jacobianhandler, which will fill the Jacobi matrix. The default implementation is to do nothing because most components do not depend on the last state.

Set the derivatives with respect to the boundary states.

If (p,qvol) = phi(rho,q), then this function must compute

dF/dState = function_derivative*phi'

and call

jacobianhandler.set_coefficient(rootvalues_index, rho_index,
dF/dState[0]); jacobianhandler.set_coefficient(rootvalues_index,
q_index, dF/dState[1]);
