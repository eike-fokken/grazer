template<typename Distribution >
double Aux::euler_maruyama_oup ( double  stability_parameter,
double  cut_off_factor,
double  last_value,
double  theta,
double  mu,
double  delta_t,
double  sigma,
Distribution &  distribution,
int  number_of_stochastic_steps 
)

Simulate a number of Ornstein-Uhlenbeck-process steps.

Uses the Euler-Maruyama-method to simulate the OUP. Note that no stability checking is done!

Parameters
stability_parameterBetween 0 and 2, the smaller, the more stable, but incurs more time steps.
cut_off_factorCut off values at mu * (1 +/- cut_off_factor)
last_valuevalue at last time step.
thetathe EM parameter for the deterministic part
muThe mean, towards which the deterministic part converges
delta_tthe time step of the outer process (not of the EM!)
sigmathe EM parameter for the stochastic part
distributionA normal distribution
number_of_stochastic_stepsThe stochastic time step is given as delta_t / number_of_stochastic_steps
Returns
The value of the process after the last step.
